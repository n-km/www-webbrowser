Sie haben das Help-Window mit "-help" geöffnet.

Wie funktioniert die Suchmaschine?

Bei dieser Suchmaschine kann man entweder nach gezielten Webseiten suchen oder die Commands verwenden und damit auf Seiten von dem Hersteller gelangen.

Wie kann ich nach gezielten Webseiten suchen?

Geben Sie einfach den Pfad der Webseite in diesem Schema ein: https://webseite.de/

Was ist bei den Commands für ein Syntax vorgegeben und welche Commands gibt es?

Wie wird die Syntax vorgegeben?

Commands sind durch ein "-" am anfang angegeben.

Welche Commands gibt es?

Bis dato gibt es:

"-help" -->zeigt diese Website an.

"-papp" -->zeigt eine Browser-Download-Site an.

"-filemanager" -->zeigt einen Filemanager an.

"-Abstimmung" -->stimmen Sie ab!

"-admin" -->Admin Anmeldung

"-cloud" -->Laden Sie etwas hoch!

"-editor" -->Schreiben Sie etwas schönes!

"-fastboot" -->Damit bleiben Sie immer schnell unterwegs!

"-launcher" -->Sie brauchen ab heute keinen Desktop mehr!

"-mobile" -->Jetzt können Sie ein Handy im Webbrowser nutzen!(Simulation)

"-mbview" -->Webseite auf dem Handy nutzen

"lite.html" -->Webseite ohne aufwendige Bilder nutzen
